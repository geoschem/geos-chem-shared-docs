#############################################
GEOS-Chem input data on AWS cloud
#############################################

The GEOS-Chem Input Data provides essential datasets for `GEOS-Chem Classic
<https://geos-chem.readthedocs.io/en/stable/>`_ and `GCHP
<https://gchp.readthedocs.io/en/stable/>`_. This includes NASA/GMAO MERRA-2 and GEOS-FP `meteorological products <https://geos-chem.readthedocs.io/en/latest/gcclassic-user-guide/input-overview.html#met>`_, `chemistry input data <https://geos-chem.readthedocs.io/en/latest/gcclassic-user-guide/input-overview.html#chemistry-input-data>`_, `emissions input data <https://geos-chem.readthedocs.io/en/latest/gcclassic-user-guide/input-overview.html#emis-inputs>`_, and other smaller datasets such as model `initial conditions <https://geos-chem.readthedocs.io/en/latest/gcclassic-user-guide/input-overview.html#initial-conditions-input-data>`_. We are pleased to announce that we are now part of the `AWS Open Data Sponsorship Program
<https://aws.amazon.com/opendata/open-data-sponsorship-program/>`_. We will continuously upload and maintain the data in an S3 Bucket for convenient access. 

In this chapter, we will describe how the data is organized and how to access the data on AWS.
